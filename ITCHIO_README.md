# VENEZUELA: RESOLVE

**A hyper-realistic geopolitical horror text adventure game**

---

## ABOUT

**Operation: Absolute Resolve** is an AI-driven interactive narrative game where you assume the role of the Venezuelan President during a 2026 US military intervention. Make life-or-death decisions in this tactical visual novel featuring psychological horror and paranoia.

---

## KEY FEATURES

- **Multi-AI Engine Support**
  - Google Gemini (Recommended)
  - SiliconFlow (Free 20M tokens!)
  - Local Mode (200+ events, offline playable)

- **Bilingual Support**
  - English (Default)
  - Simplified Chinese

- **Dynamic Narrative System**
  - Real-time AI-generated storylines
  - "Doomsday Clock" mechanics
  - Multiple endings (Victory, Death, Prison, Exile, Mental Breakdown, Coup)

- **Real-time Stats Tracking**
  - Public Support
  - Military Loyalty
  - Security Level
  - Panic Index

- **Immersive UI Design**
  - Military tactical interface
  - Typewriter text effects
  - Decryption animations
  - Glitch visuals based on panic level

### Offline Mode

- Select "Local" tab
- No internet or API key required
- 200+ handcrafted events with smart rules engine

## GAME MECHANICS

### Phase System

1. **Political Maneuvering (Turn 1-5)**
   - Perimeter breach, chaos begins
   - Cyber attacks and ghost signals

2. **Military Siege (Turn 6-15)**
   - Enemy enters building
   - Close-quarters combat and betrayal

3. **Escape and Evasion (Turn 16+)**
   - Last stand or escape attempt
   - Desperate moments

### Decision Types

- **Aggressive**: High-risk, high-reward military actions
- **Diplomatic**: Negotiate solutions
- **Stealth**: Avoid direct conflict
- **Desperate**: Last-resort choices
