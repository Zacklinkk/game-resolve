## 🎛️ 音频实现建议

### **代码实现**（React + Howler.js）：

```bash
# 安装音频库
npm install howler
```

```typescript
// services/audioManager.ts
import { Howl } from 'howler';

export const playMusic = (trackName: string) => {
  const sound = new Howl({
    src: [`/audio/gameplay/${trackName}.mp3`],
    loop: true,
    volume: 0.5,
    fade: true
  });
  sound.play();
};

export const playStinger = (eventName: string) => {
  const stinger = new Howl({
    src: [`/audio/events/${eventName}.mp3`],
    volume: 0.7
  });
  stinger.play();
};
```

### **音量控制建议**：
- 音乐：40-60%
- 音效：70-80%
- 警报/提示音：50%

## 🔄 Suno 生成流程

### **Step 1**: 访问 https://suno.ai

### **Step 2**: 选择 "Custom Mode"

### **Step 3**: 粘贴提示词

**示例**（主菜单音乐）：
```
[Instrumental]
Dark ambient military atmosphere, deep pulsing bass drones, distant helicopter
rotors, ticking clock rhythm, ominous synthesizer pads, paranoid tension,
geopolitical thriller, minimalist, slow tempo 60 BPM, D minor, electronic
industrial elements, suspense building, cinematic quality, professional mixing
```

### **Step 4**: 设置参数
- **Length**: 2:00 (循环音乐最佳时长)
- **Instrumental**: ✅ (必须勾选)
- **Style**: Custom (自定义)

### **Step 5**: 生成并选择最佳版本
- 生成 2-4 个版本
- 选择最符合游戏氛围的
- 下载高品质 MP3 或 WAV

## 🎵 额外建议

### **音量淡入淡出**：
```javascript
// 场景切换时淡出当前音乐，淡入新音乐
sound.fade(0.5, 0, 1000); // 1秒淡出
```

### **音乐动态调整**：
```typescript
// 根据恐慌指数调整音乐
const adjustMusicByPanic = (panicLevel: number) => {
  if (panicLevel > 80) {
    // 添加更多不和谐音、加快节奏
    playIntenseVariant();
  }
};
```

### **循环点标记**：
- 确保音乐无缝循环
- 在音频编辑软件中标记循环点
- 导出时设置循环元数据
